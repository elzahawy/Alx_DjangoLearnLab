# Authentication System

## Features
- User registration with email
- Login and logout
- Profile management (update email)
- CSRF protection and password hashing included

## Setup
1. Run `python manage.py migrate`
2. Start the server: `python manage.py runserver`
3. Access pages:
   - Register: /register/
   - Login: /login/
   - Profile: /profile/
   - Logout: /logout/

## Testing
- Try registering, logging in, updating profile email, and logging out.
